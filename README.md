# Movie Shop Assessment (MERN Stack)

A full-stack movie management application built using the MERN stack, designed with scalability, performance, and best practices in mind.
The system supports asynchronous movie insertion using a distributed queue, secure APIs, and a modern frontend UI.

Project URL - https://cool-conkies-c1913d.netlify.app/

Frontend (React)
- Tech Stack
- React (Vite)
- Material UI (MUI v6)
- Axios
- React Hooks

Features
- Add new movies with validation
- Edit existing movies
- List movies with pagination, search, and sorting
- Client-side form validation (UX friendly)
- Clean, responsive UI using MUI
